# Massachusetts Champion & Legacy Trees

Interactive dashboard mapping every Champion and Legacy tree in the Massachusetts DCR registry.

**[View the dashboard →](https://jdinino.github.io/massachusetts-legacy-trees/)**

## Features

- Leaflet map with clustered markers — stars for Champions, dots for Legacy trees
- Cascading filters by species, county, city, year, height, and champion points
- Sortable full registry table with address, GPS coordinates, and Google Maps links
- County breakdown, species browser, and measurement timeline
- Estimated tree ages based on circumference and ISA species growth factors
- Species reference images from Wikipedia
- Street View links for each tree location

## Data

Built from the DCR Bureau of Forest Fire Control and Forestry's Champion Trees and Legacy Trees spreadsheets. All data is publicly available through the [Massachusetts Legacy Tree Program](https://www.mass.gov/guides/massachusetts-legacy-tree-program).

Tree locations are best-guess geocoded from city, address, and landmark data — not surveyed GPS positions.

Estimated ages use standard arboriculture growth factors (circumference ÷ π × species growth rate) and should be treated as rough approximations.

## Nominate a Tree

Know a big, historic, or remarkable tree in Massachusetts? You can nominate it through the [DCR Legacy Tree Program](https://www.mass.gov/guides/massachusetts-legacy-tree-program). A trained forester will visit to verify species and measurements.
